## 1. ""eaglei_coverage_by_NERC.csv" is the EAGLE-I utility customer coverage data aggregated at the NERC level. 
## 2. "eaglei_NERC_time_adjusted.csv" is the EAGLEI-I outage data aggregated at the NERC level with time zone adjusted to local time. 
## 3. "eaglei_rolling_reliability.csv" contains the calculated reliability mertrics (i.e., SAIFI, SAIDI, and CAIDI) for the EAGLEI-I dataset based on 365-day rolling average method.
## 4. "IEEE_rolling_reliability.csv" contains the calculated reliability metrics (i.e., SAIFI, SAIDI, and CAIDI) for the IEEE outage datase based on 365-day rolling average method. 
## 5. "IEEE_rolling_reliability_MED_excluded.csv" contains the calculated reliability metrics (i.e., SAIFI, SAIDI, and CAIDI) for the IEEE outage datase excluding Major Event Days (MED) based on 365-day rolling average method.
